TrafficOps API API Reference

The following REST APIs are used by the Traffic Portal UI along with several Traffic Control backend component and services.

This documentation is fully compliant with the Swagger 2.0 specification

API Endpoint
https://localhost:8443/api/1.3
Contact: dev@trafficcontrol.incubator.apache.org
Request Content-Types: application/json
Response Content-Types: application/json
Schemes: http, https
Version: 1.3

Schema Definitions

ASN: object

asn: integer (int64)

The ASN to retrieve

Autonomous System Numbers per APNIC for identifying a service provider

cachegroup: string

Related cachegroup name

cachegroupId: integer (int64)

Related cachegroup id

id: integer (int64)

ID of the ASN

lastUpdated: string

LastUpdated

Example
{
  "asn": "integer (int64)",
  "cachegroup": "string",
  "cachegroupId": "integer (int64)",
  "id": "integer (int64)",
  "lastUpdated": "string"
}

ASNResponse: object

A Single ASN Response for Update and Create to depict what changed

response: ASN
Example
{
  "response": {
    "asn": "integer (int64)",
    "cachegroup": "string",
    "cachegroupId": "integer (int64)",
    "id": "integer (int64)",
    "lastUpdated": "string"
  }
}

ASNsResponse: object

A List of ASNs Response

response: ASN

in: body

Example
{
  "response": [
    {
      "asn": "integer (int64)",
      "cachegroup": "string",
      "cachegroupId": "integer (int64)",
      "id": "integer (int64)",
      "lastUpdated": "string"
    }
  ]
}
